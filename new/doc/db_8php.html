Get data about invert search on a dictionary (currently FAS)

Parameters
$methodParamLetter, part of word or range according to the method, that results words must begin
$filterFilter options (actually used there for get method)
Returns
Array with that for each word :
Index 1 : The word
Index 2 : The absolute number of occurrences of this reaction.
Index 3 : The string to display on the website
Index 4 : Word counter (unused here)
Index 5 : The number of stimuli that triggered it.

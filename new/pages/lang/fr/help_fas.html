Dictionnaire direct

Si vous n'entrez qu'une seule lettre, vous verrez tous les stimuli commençant par cette lettre (dans l'ordre alphabétique).
Le champ de réponse montre toutes les réponses données à ce stimulus dans l'ordre inverse de la fréquence.
Les chiffres entre parenthèses indiquent :

  1. le nombre total de réponses données à un stimulus donné,
  2. le nombre de réactions différentes,
  3. le taux d'échec,
  4. le taux de réponse unitaire .

Dictionnaire inversé

Le dictionnaire inversé présente les réactions obtenues au cours de l'expérience et les stimuli, qui les ont suscités par ordre décroissant de fréquence. Vous pouvez choisir dans quel ordre vous voulez placer les réactions :

  • Par ordre alphabétique : Toutes les réactions commençant par le caractère (ou suite de caractères) sélectionné(es),
  • Par nombre de stimuli : Toutes les réponses sont triées par le nombre de stimuli auxquels un mot particulier a été donné en réponse.
  • Par nombre de réponse : Toutes les réponses sont triées par le nombre de réponse.
    • Les chiffres entre parenthèses indiquent :
    1. fréquence des réponses,
    2. le nombre de stimuli auxquels un mot donné a été donné en réponse.

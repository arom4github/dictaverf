Dictionnaire direct

Si vous saisissez une seule lettre, vous verrez tous les stimuli commençant par cette lettre (par ordre alphabétique).
Le champ réponse contient toutes les réponses données à ce stimulus, dans l'ordre inverse de la fréquence (indiqué par le numéro immédiatement après la réponse). Dans la première colonne, toutes les réactions, quelle que soit leur origine géographique Dans les autres colonnes, les réactions sont réparties en fonction de l'origine géographique (France, Belgique, Suisse, Canada). La couleur indique le fait de la coïncidence d'une réaction donnée dans deux ou plusieurs zones.
Les chiffres entre parenthèses indiquent :

  1. le nombre total de réponses données à un stimulus donné,
  2. le nombre de réponses différentes [les crochets quadratiques indiquent le pourcentage de réponses différentes par rapport au total : plus le pourcentage est faible, plus les réponses des répondants sont uniformes].
  3. le taux d'échec,
  4. le taux de réponse unitaire.
Dans la colonne des stimuli, il est possible de cliquer sur un bouton pour afficher une courbe montrant le nombre total de réactions et par pays (pas encore mis en œuvre).
